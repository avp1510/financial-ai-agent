# Financial AI Agent

A smart AI system that helps you analyze stocks and get financial insights. It combines real-time stock data with web search to give you comprehensive market information.

## Features

- Get real-time stock prices and fundamentals
- Read analyst recommendations and company news
- Search the web for financial context and market trends
- Smart multi-agent system that combines different data sources
- Built with reliability in mind - handles errors gracefully

## Quick Start

1. Install dependencies: `pip install -r requirements.txt`
2. Add your API keys to `.env` file
3. Run the web interface: `python playground.py`
4. Or try the command line: `python financial_agent_oop.py`

## Example Queries

- "What's the current price of AAPL?"
- "Get analyst recommendations for TSLA"
- "Summarize recent news about NVIDIA"
- "Compare AAPL and MSFT fundamentals"
